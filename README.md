![](./docs/img/preview.png)

## Install ##

```bash
npm install -g movoto-cli
```
> Use `sudo` if you are working on unix like operating system

## Usage ##

```bash
movoto
```

### Options ###

* `--version` Print the current `movoto-cli` version
* `--help`  Print `movoto-cli` help menu
